## 常见问题

### Q1: 为什么要使用虚拟环境？
A: Python 3.13 在 macOS 上通过 Homebrew 安装后，系统不允许直接安装第三方库。虚拟环境可以隔离项目依赖，避免影响系统 Python。

### Q2: 如何检查虚拟环境是否激活？
A: 激活后，终端提示符前会显示 `(venv)`。

### Q3: 如果忘记激活虚拟环境会怎样？
A: 会出现 `ModuleNotFoundError` 错误，提示找不到 numpy 等库。

### Q4: 如何安装其他库？
A: 先激活虚拟环境，然后使用 `pip install 库名`。

### Q5: 虚拟环境占用多少空间？
A: 约 200-300 MB。
