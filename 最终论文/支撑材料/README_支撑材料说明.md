## 📊 数据文件说明

### 1. USDT_USDC_对比表.csv
**内容**：USDT和USDC在各个评价指标上的详细评分对比
- **字段**：指标名称、USDT评分、USDC评分
- **用途**：问题一的量化分析基础
- **引用位置**：论文第4节

### 2. 资产配置方案对比.csv
**内容**：保守型、平衡型、激进型三种策略的详细配置方案
- **字段**：策略类型、各资产配置比例、预期收益、波动率、流动性、夏普比率
- **用途**：问题二的优化结果
- **引用位置**：论文第5节

### 3. 预测结果汇总.csv
**内容**：2026-2030年稳定币市场规模预测数据
- **字段**：年份、美元稳定币市值、非美元稳定币市值、市场份额
- **用途**：问题三的预测分析
- **引用位置**：论文第6节

### 4. 货币主权风险评估结果.csv
**内容**：30个国家的货币主权评分和相关指标
- **字段**：国家名称、货币主权评分、稳定币指数、通胀率、外币存款比例等
- **用途**：问题四的货币主权分析
- **引用位置**：论文第7节

### 5. 高风险国家报告.csv
**内容**：识别出的高风险国家详细信息
- **字段**：国家、风险等级、主权丧失概率、关键风险指标
- **用途**：问题四的风险预警
- **引用位置**：论文第7节

### 6. 政策简报_20251105.txt
**内容**：完整的政策简报文本
- **包含**：核心摘要、SWOT分析、成本效益分析、政策建议等
- **用途**：问题五的综合分析
- **引用位置**：论文第8节

## 🔬 计算过程说明

### 问题一：AHP权重计算
1. 构建4×4判断矩阵
2. 计算最大特征值λ_max = 4.089
3. 一致性检验：CR = 0.033 < 0.10（通过）
4. 归一化得到权重向量：[0.30, 0.25, 0.25, 0.20]

### 问题二：多目标优化
1. 定义目标函数：收益最大化、风险最小化、流动性最大化
2. 设置约束条件：预算约束、流动性约束、资产比例约束
3. 使用SLSQP算法求解，得到三种策略的最优配置
4. 进行压力测试和场景分析验证稳健性

### 问题三：时间序列预测
1. 多元回归识别关键影响因素（R² = 1.0000）
2. ARIMA(2,1,2)模型预测短期趋势
3. Logistic模型拟合长期增长（K = 17,393亿）
4. 50%-50%权重组合得到最终预测

### 问题四：Logit预警模型
1. 构建货币主权评价指标体系（5个维度）
2. Pearson相关性分析（ρ = -0.849，p < 0.001）
3. Logit回归预测主权丧失概率（准确率93.3%）
4. 识别主权丧失概率>40%的高风险国家

### 问题五：综合分析
1. SWOT加权评分（S=8.10, W=5.35, O=8.05, T=6.25）
2. NPV计算（贴现率5%，10年期）
3. RWA市场规模预测（CAGR 91.0%）
4. 提出7条具体政策建议
